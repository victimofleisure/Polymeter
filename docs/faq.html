How can I support this project?

  • Go to the reviews page and recommend Polymeter (just press the "thumbs up" button), or better yet, review it. To review it, you'll need a SourceForge account; register here.
  • Recommend Polymeter to friends, promote it on forums, etc. Polymeter competes with many excellent and well-established applications, so word of mouth is crucial.
  • Donate to the project. Free software developers are chronically underpaid!
  • Contribute to the project, e.g. by coding, testing, documenting, or translating.

Why isn't check for updates working?

If attempting to check for updates gives the error "Can't download file: A connection with the server could not be established" it may be a TLS version issue. In Control Panel / Internet Options, Advanced tab, under Security, find the options "Use TLS 1.1" and "Use TLS 1.2" and make sure they're both checked. In Windows 7, only TLS 1.0 is checked by default.

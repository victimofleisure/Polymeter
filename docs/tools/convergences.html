Convergence Calculator

This script finds all the convergences between a given set of positive integers. It can be useful when composing polymeter or phase music, because it shows how loops of different lengths synchronize over time. The implementation calculates the unique Least Common Multiples for all possible subsets of the input set having two or more values.

This software is a specialized MIDI sequencer for composing polymeter, which is music that uses multiple meters simultaneously. Each track has its own loop length, and when the lengths differ, the tracks "slip" (or shift phase) relative to each other. The resulting interference patterns are sufficiently intricate that variations similar to a live performer's embellishments can be generated algorithmically.

The Polymeter MIDI Sequencer evolved from software I developed in the 1990s in order to compose polymeter electronic dance music.
